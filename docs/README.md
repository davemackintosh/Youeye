# youeye design docs

Living record of the design decisions that shape this project. These mirror the per-project Claude memory but are version-controlled so they travel with the repo.

- [overview.md](overview.md) — goals, hard constraints, non-goals, SVG-as-truth commitment.
- [stack.md](stack.md) — chosen Rust crates and workspace layout.
- [features.md](features.md) — scoped features and the two distinctive differentiators (rulers-as-entities, parametric design system).
- [status.md](status.md) — current phase, task list, next steps. **Update after each working session.**

Dates on decisions are absolute so they remain interpretable over time.
